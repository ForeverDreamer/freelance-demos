# sample_data

Synthetic documents for demoing the RAG pipeline. No real product,
company, or customer data.

- `company_handbook.md`  fictional employee policies (PTO, remote work, expenses)
- `product_faq.md`  fictional SaaS product FAQ (pricing, limits, billing)
- `api_reference.md`  fictional REST API reference (auth, endpoints, errors)

Drop additional `.md`, `.txt`, or `.pdf` files here and re-run
`ragchat.py ingest ./sample_data` to index them.
